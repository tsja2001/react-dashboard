# 可视化大屏项目

## 命令

### 安装依赖 `pnpm install`

### 启动项目 `pnpm dev`

### 打包项目 `pnpm build`

## 前期工作(建议)

### 代码检查 vscode 安装 eslint 插件

### 代码格式化 vscode 安装 prettier 插件

### css module 跳转 vscode 安装 css modules 插件
